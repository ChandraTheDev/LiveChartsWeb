Inverted Series

Inverting series in LiveCharts is practically automatic, you must use the inverted version of the series, then live charts will handle the inverted draw, notice the configuration for Vertical and Horizontal series is different, and you must concider this if plotting a custom type.

Practically all series (but financial) have a built in inverted version, for example the LineSeries class inverted version is VerticalLineSeries class, in The cartesian chart article you could find a full list with the supported series.

Code Behind

using System;
using System.Windows.Forms;
using LiveCharts;
using LiveCharts.Wpf;
namespace Winforms.Cartesian.Inverted_Series
{
    public partial class InvertedSeries : Form
    {
        public InvertedSeries()
        {
            InitializeComponent();
            cartesianChart1.Series.Add(new VerticalLineSeries
            {
                Values = new ChartValues<double> { 3, 5, 2, 6, 2, 7, 1 }
            });
            cartesianChart1.Series.Add(new RowSeries
            {
                Values = new ChartValues<double> {6, 2, 6, 3, 2, 7, 2}
            });
            cartesianChart1.AxisY.Add(new Axis
            {
                Separator = new Separator { Step = 1}
            });
            cartesianChart1.AxisX.Add(new Axis
            {
                MinValue = 0
            });
            cartesianChart1.DataTooltip.SelectionMode = TooltipSelectionMode.SharedYValues;
        }
    }
}
