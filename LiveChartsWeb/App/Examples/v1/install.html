Installation

Note: images included in this article use Visual Studio 2015, but it is really similar for another visual studio version.

Install package from NuGet, go to Solution Explorer, right click on references, then Manage NuGet Packages…

Browse for LiveCharts, select the package and click on install, that's it you are ready to use this library.

Don't forget to add name space using LiveCharts;

Or XAML namespace if using wpf xmlns:lvc="clr-namespace:LiveCharts;assembly=LiveCharts"

If you are using WinForms you should see LiveCharts controls in your Toolbox
