Multiple Axes Chart

There are some cases when you need to compare the trend between multiple series but they have a different scale or different measure unit then you can create another axis and style it according to it's series.

Notice how each color set of series has a really big numeric diference but they scale according to a different axis. 

<lvc:LineChart>
  <lvc:LineChart.Series>

    <!--using default axis-->
    <lvc:LineSeries Title="A" Values="300, 600, 200" Stroke="Violet" Fill="Transparent" />

    <!--using ComplementaryAxes[0]-->
    <lvc:LineSeries Title="B" Values="0, 0, 0" Stroke="Chocolate" 
                 Fill="Transparent" FromComplementaryY="0" />
    <lvc:LineSeries Title="C" Values="10, 10, 10" Stroke="Chocolate" 
                 Fill="Transparent" FromComplementaryY="0" />

    <!--using ComplementaryAxes[1]-->
    <lvc:LineSeries Title="D" Values="20000, 20000, 20000" Stroke="Navy" 
                 Fill="Transparent" FromComplementaryY="1" />
  </lvc:LineChart.Series>
  <lvc:LineChart.AxisY>
    <lvc:Axis Foreground="Violet" Title="Violet Axis">
      <lvc:Axis.ComplementaryAxes>
        <lvc:Axis Title="Chocolate Axis" Foreground="Chocolate" />
        <lvc:Axis Title="Navy Axis" Foreground="Navy" />
      </lvc:Axis.ComplementaryAxes>
    </lvc:Axis>
  </lvc:LineChart.AxisY>
</lvc:LineChart>

Notice that we created a custom Axis and set the tile property to Violet Axis then added 2 extra axis: Chocolate Axis and Navy Axis.

To plot series "B" and "C" in chocolate axis we used the FromComplementaryY property which means that the Y values in this both series will scale in the first axis in ComplementaryAxes property (ComplementaryAxes[0]), the same way "D" series uses ComplementaryAxes[1] in this case the Navy Axis.

Finally "A" series uses the default chart Axis, the Violet Axis.

Notice all series scales the same on X, if you need a different scale on X you must use the FromComplementaryX prperty.
