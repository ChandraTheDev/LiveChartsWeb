Installation

Note: images included in this article use Visual Studio 2015, but it is really similar for another visual studio version.

Install package from NuGet, go to Solution Explorer, right click on references, then Manage NuGet Packages…

Browse for LiveCharts.Wpf, select the package and click on install

Add namespace to your XAML

xmlns:lvc="clr-namespace:LiveCharts.Wpf;assembly=LiveCharts.Wpf"

And to code behind if necessary

using LiveCharts;
using LiveCharts.Wpf;
