Installation

Note: images included in this article use Visual Studio 2015, but it is really similar for another visual studio version.

Install package from NuGet, go to Solution Explorer, right click on references, then Manage NuGet Packages…

Browse for LiveCharts, select the package and click on install.

Now we need to add some extra references, Go to Solution Explorer -> right click on Add Reference

A dialog like next image will appear, go to Assemblies -> Framework and then search for the next components

  • PresentationFramework
  • PresentationCore
  • WindowsBase
  • System.Xaml

In the Windows Forms Designer, open your form. In the Properties window, click Events, and then double-click the Load event to create an event handler. The Code Editor opens to the newly generated event handler. Replace the code in the Load event with the next block, the Load event handler creates an instance of LineChart and adds it to the ElementHost control's collection of child controls. The ElementHost control is added to the form's collection of child controls. 

// Create the ElementHost control for hosting a LineChart
ElementHost hst = new ElementHost();
host.Dock = DockStyle.Fill;

// Create a new LineChart.
var uc = new LiveCharts.LineChart();

// Assign the Chart to the ElementHost control's Child property.
host.Child = uc;

// Add the ElementHost control to the form's
// collection of child controls.
this.Controls.Add(host);

you may also want to LiveCharts name space. 

using LiveCharts;
