Basics

First of all LiveCharts is designed to be easy for the user, everything is updated and animated automatically, it also handles the performance, feel free to change 1 million points in 1 second, the library will not update 1 million times, it will only do it when it concider it is necessary, when you add/remove series, or add/remove values the chart will update by itself, you really dont need to worry about anything but your bussiness, let LiveCharts handle the charting, also implementing IObservableChartPoint will also update the chart when a property changes, there are many classes already defined and ready to use that implement IObservableChartPoint

The next image will guide you to get more familiar with LiveCharts.

All the series have a Stroke and Fill properties.

By default if you don't set the Fill or Stroke properties, LiveCharts will use a predifined color, according to the series positions in the Chart.SeriesCollection property, you can easly define your own colors set, this colors will repeat if necessary. 

LiveCharts.Wpf.Charts.Chart.Chart.Colors = new List<System.Windows.Media.Color>
{
  System.Windows.Media.Colors.Red,
  System.Windows.Media.Colors.Blue,
  System.Windows.Media.Colors.Green
};

Or you can simply set a Fill and Stroke only for a specific series 

mySeries.Stroke = System.Windows.Media.Brushes.Red;
mySeries.Fill = System.Windows.Media.Brushes.Blue;

Also setting the Series.Visibility (will define the drawn shape visibility), Panel.ZIndex (drawn shape z-index property), Series.StrokeDashArray (dashed strokes) properties will be binded to the drawn shape, for example: 

mySeries.Visibility = System.Windows.Visibility.Hidden;
mySeries.StrokeDashArray = new System.Windows.Media.DoubleCollection {2};
System.Windows.Controls.Panel.SetZIndex(mySeries, 3);

If necessary you can also define your own Tooltips o Legends controls, create a new UserControl that inherits from DefaultTooltip or DefaultLegend, and do what ever you want with it. Then set the Chart.DataTooltip or Chart.ChartLegend properties to a new instance of your control, notice this should be WPF control. 

myChart.DataTooltip = new MyCustomTooltop();
