Labels

A Label is any string representation of any value in the chart, they are normally placed over the axis length and in tooltips.

There are 2 types of labels in LiveCharts, Formatted and Mapped labels.

Formatted Labels

A Formatted label is useful when there is a direct conversion between the chart value and the label, for example, in the image above, the Y axis has a range of values between 8 and 26, but because of the current formatter, we are able to see 8 as 8.00k items.

This is really easy to achieve, the Axis.LabelFormatter property, stores a function that takes a double value as parameter and returns a string, LiveCharts will use this function every time it needs to display a chart value as string. 

MyAxis.LabelFormatter = val => val.ToString("C"); //as currency
MyAxis.LabelFormatter = val => val + "°"; //as degrees
MyAxis.LabelFormatter = val => val + ".00 items sold"; //or any other custom format

Or follow MVVM pattern: 

//declare a public property in your view.
public Func<double, string> YFormatter { get; set; }

//in the constructor or in the Load Event of your view or model.
YFormatter = val => val + ".00 items sold";

//even faster with C#6
//public Func<double, string> YFormatter { get; set; } = val => val + ".00 items sold";

Now let's bind it to your Axis LabelFormatter property: 

<lvc:BarChart >
    <lvc:BarChart.AxisY>
        <lvc:Axis Name="MyAxis" Title="Sold Items" LabelFormatter="{Binding YFormatter}">
        </lvc:Axis>
    </lvc:BarChart.AxisY>
</lvc:BarChart>

You can find the full example at Github https://github.com/beto-rodriguez/Live-Charts/tree/master/WPFExamples/BarExamples/Mvvm

Mapped Labels

A mapped label is normally useful to map a position with a name, for example when first point belongs to John second to Susan and third one to Charles. 

<lvc:BarChart>
    <lvc:BarChart.AxisX>
        <lvc:Axis Title="Month"
                     Labels="John, Susan, Charles"/>
    </lvc:BarChart.AxisX>
</lvc:BarChart>

To show an example we are going to talk about an X indexed chart, this does not mean you cannot label Y axis, to Label a Y axis you would normally need to invert the chart first.

Axis.Labels type is IList<string>, and by default X axis is indexed, this means first point is 0, second is 1, third is 2... and so on, this means and both x values and IList share an index for each position.

So when x = 0 then label will be Axis.Labels[0], when x = 1 Labels[1], x = 2 then labels[2], x = n then labels[n].

Notice that if X index is greater than Labels.Count an empty string will be returned.

Axis.Labels hides Axis.LabelFormatter, mean while Axis.Labels is not null then labels will be pulled from Axis.Labels, if Axis.Labels is null then LiveCharts will use the formatter, if both are null then the raw value will be used as label.
