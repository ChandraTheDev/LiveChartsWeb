The Cartesian Chart

The Cartesian Chart class allows you to plot any series that uses a Cartesian coordinate system, every point is a pair of values (X, Y), then how is LiveCharts able to plot ChartValues<double> if I only passed a value? simple, in simple terms Y will be the value you passed, and X the index of the value in the array, if you want to learn more about how LiveCharts works, you can read this.

The Cartesian chart supports many series, you can combine any of these series in a Cartesian chart:

Line Series
Vertical Line Series
Column Series
Row Series
Stacked Area Series
Vertical Stacked Area Series
Stacked Column Series
Stacked Row Series
Heat Series
OHCL Series (Financial)
Bubbles Series

Here is an example using every of these series, notice some series use a specialized point, LiveCharts already knows how to plot many types, if you need to define your own, please take a read here, it is pretty easy. 

using LiveCharts;
using LiveCharts.Defaults; //Contains the already defined types.

// ...

LiveCharts.SeriesCollection series = new LiveCharts.SeriesCollecion 
{
  new LineSeries
  {
    //The ObservableValue class notifies the chart to update when value changes
    Values = new ChartValues<LiveCharts.Defaults.ObservableValue>
    {
        new LiveCharts.Defaults.ObservableValue(4),
        new LiveCharts.Defaults.ObservableValue(4),
        // ...
    }
  },
  new VerticalLineSeries
  {
    //if you need to specify both, X and Y then use a point instead
    //otherwise the another coordinate will be a zero-based index
    Values = new ChartValues<ObservablePoint>
    {
        new ObservablePoint(3, 6),
        new ObservablePoint(6, 2),
        // ...
    }
  },
  new ColumnSeries
  {
    Values = new ChartValues<ObservableValue>
    {
      new ObservableValue(4),
      new ObservableValue(2),
      // ...
    }
  },
  new RowSeries
  {
    Values = new ChartValues<decimal>
    {
      7,
      4,
      // ...
    }
  },
  new StackedAreaSeries
  {
    Values = new ChartValues<double>
    {
      5,
      3,
      // ...
    }
  },
  new VerticalStackedAreaSeries
  {
    Values = new ChartValues<float>
    {
      5,
      3,
      // ...
    }
  },
  new StackedColumnSeries
  {
    Values = new ChartValues<ObservableValue>
    {
      new ObservableValue(5),
      new ObservableValue(8),
      // ...
    }
  },
  new StackedRowSeries
  {
    Values = new ChartValues<ObservableValue>
    {
      new ObservableValue(5),
      new ObservableValue(8),
      // ...
    }
  },
  new OhlcSeries
  {
    Values = new ChartValues<OhlcPoint>
    {
      new OhlcPoint(32, 35, 30, 32),
      new OhlcPoint(33, 38, 31, 37),
      // ..
    }
  },
  new BubbleSeries
  {
    Values = new ChartValues<BubblePoint>
    {
      new BubblePoint(5, 5, 20),
      new BubblePoint(3, 4, 80), 
      // ...
    }
  }
}
