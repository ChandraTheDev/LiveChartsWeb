Series Colors

If you don't specify a color for a series, then there is a default colors collection in the static Chart.Colors property that stores the default colors for all charts series, when a chart uses all colors in this collection, then colors will repeat. All series have a Stroke and Fill properties, the fill property by default has certain transparency according to series type.

The default colors collection is based on material design colors, but if you need to secify your own set, then when your app starts you need to change the Chart.Colors property, this will be enough to initialize all your charts with this colors set. 

Chart.Colors = new List<Color>
{
    Colors.Red,
    Colors.Green,
    Colors.Blue
};
