Series Colors

If you don't specify a color for a series, then there is a default colors collection in the static Chart.Colors property that stores the default colors for all charts series, when a chart uses all colors in this collection, then colors will repeat. All series have a Stroke and Fill properties, the fill property by default has certain transparency according to series type.

The default colors collection is based on material design colors, but if you need to secify your own set, then when your app starts you need to change the Chart.Colors property, this will be enough to initialize all your charts with this colors set. 

Chart.Colors = new List<Color>
{
  Colors.Red,
  Colors.Green,
  Colors.Blue
};

Of course you could only need to set the color of a specific series, Stroke and Fill properties type is Brush, this means you can easily use complex fills and strokes like images, gradients or just a simple solid color. 

<lvc:BarChart>
  <lvc:BarChart.Series>
    <lvc:BarSeries Title="Maria" Values="1, -4, -4, 6, -5.5, 6, 6, -7"
                      Stroke="#FFB300" Fill="#AAFFB300" StrokeThickness="4" />
      <lvc:BarSeries Title="John" Values="-3.0, 3.5, -4.3, 6.8, 6.5 ,-7, -5.5, -6"
                        Stroke="#FFA000" StrokeThickness="4">
        <lvc:BarSeries.Fill>
          <LinearGradientBrush>
            <GradientStop Color="#FFB30000" Offset="0" />
            <GradientStop Color="#FF8F00" Offset="1" />
          </LinearGradientBrush>
        </lvc:BarSeries.Fill>
      </lvc:BarSeries>
      <lvc:BarSeries Title="Erick" Values="-4.0, 4.2, -5.0, 7.0, -5.0, 5.5, -4.4, -7.3"
                        Stroke="#FF8F00"
                        StrokeThickness="4">
        <lvc:BarSeries.Fill>
          <ImageBrush ImageSource="../../imagefill.jpg"></ImageBrush>
        </lvc:BarSeries.Fill>
      </lvc:BarSeries>
  </lvc:BarChart.Series>  
</lvc:BarChart>
