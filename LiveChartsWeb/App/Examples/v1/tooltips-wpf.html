Tooltips

Tooltips are shown when the user moves the mouse over a data point, by default tooltips are enabled, to disable them all you only need to set the Chart.Hoverable property to false.

You can create your own tooltips when you need to display more information or when you just need another visual style.

We are going to build a custom tooltip, in this case we will display another property (Rentability), to create a custom tooltip, add a new WPF user control to your project, and replace the XAML with the next code 

<tooltip:IndexedTooltip x:Class="ChartsTest.BarExamples.Mvvm.SalesTooltip"
                           xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
                           xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
                           xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
                           xmlns:d="http://schemas.microsoft.com/expression/blend/2008"
                           xmlns:tooltip="clr-namespace:LiveCharts.Tooltip;assembly=LiveCharts"
                           xmlns:mvvm1="clr-namespace:ChartsTest.BarExamples.Mvvm"
                           mc:Ignorable="d"
                           d:DataContext="{d:DesignInstance tooltip:DefaultIndexedTooltip}">
  <tooltip:IndexedTooltip.Resources>
    <mvvm1:RentabilityConverter x:Key="RentConverter"></mvvm1:RentabilityConverter>
  </tooltip:IndexedTooltip.Resources>
  <Border CornerRadius="2">
    <Border.Background>
      <SolidColorBrush Color="WhiteSmoke" Opacity=".85" />
    </Border.Background>
    <StackPanel Margin="4">
      <ItemsControl ItemsSource="{Binding Data}" Grid.IsSharedSizeScope="True">
        <ItemsControl.ItemTemplate>
          <DataTemplate DataType="{x:Type tooltip:IndexedTooltipData}">
            <Grid Margin="2">
              <Grid.ColumnDefinitions>
                <ColumnDefinition />
                <ColumnDefinition Width="Auto" SharedSizeGroup="Title" />
                <ColumnDefinition Width="Auto" SharedSizeGroup="Value" />
                <ColumnDefinition Width="Auto" SharedSizeGroup="Highest" />
             </Grid.ColumnDefinitions>
             <Ellipse Width="11" Height="11" StrokeThickness="1" Stroke="{Binding Stroke}" Fill="{Binding Fill}" />
             <TextBlock Grid.Column="1" Text="{Binding Series.Title}" Margin="4 0" VerticalAlignment="Center" Foreground="#404040" />
             <TextBlock Grid.Column="2" Text="{Binding Value}" Margin="4 0" VerticalAlignment="Center" Foreground="#404040" />
             <TextBlock Grid.Column="3" Text="{Binding Point, Converter={StaticResource RentConverter}}" Margin="4 0" VerticalAlignment="Center" Foreground="#404040"></TextBlock>
           </Grid>
         </DataTemplate>
       </ItemsControl.ItemTemplate>
      lt;/ItemsControl>
      <TextBlock Text="{Binding Header}" HorizontalAlignment="Center" FontWeight="SemiBold" Foreground="#404040" />
    </StackPanel>
  </Border>
</tooltip:IndexedTooltip>

Notice how the tooltip inherits from LiveCharts.IndexedTooltip, in this case the code behind of this tooltip is empty, we are only using a converter to cast Point.Instance to our view model and then return the Rentability property as string. The next block shows the converter 

public class RentabilityConverter : IValueConverter
{
  public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
  {
    var point = value as ChartPoint; 
    if (point == null) return "";
    var salesData = point.Instance as SalesData;
    return salesData == null
           ? ""
           : salesData.Rentability.ToString("P");
  }

  public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture
  {
    throw new NotImplementedException();
  }
}
