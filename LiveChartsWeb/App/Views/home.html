Have you ever tired to buid a modern .net application?, if so maybe you have used one of these awesome frameworks: ReactiveUi, mahapps.metro or Material Design In XAML Toolkit everything seems fine, but then you need data visualization and you notice there are no modern and free charting controls for .net, how could this be possible? LiveCharts is much more than beauty, responsive and animated charts, they are also totally automatic, you just add/remove values or series and chart refreshes. you really dont need to worry about anything, have you ever used a generic list? like List<double> myList = new List<double>(); if so then you are ready to use LiveCharts, this project is Mvvm orientated and licenced under MIT licence, right now tested only in WPF and WindowsForms.

Install package from NuGet

                Install-Package LiveCharts
                
Add namespace to your XAML if using WPF

                xmlns:lvc="clr-namespace:LiveCharts;assembly=LiveCharts" 


                Or WinForms and codebehind 


                using LiveCharts;

                
That's it, you are ready
