How does LiveCharts works?

Livechars core components are Chart, SeriesCollection, Series and IChartValues, all charts inherit from chart and have a Series property of type SeriesCollection, this means that practically everything that works for a line chart, will work for bar charts, or any other chart type. SeriesCollection class tracks when you add/remove a series from a chart then notifies the chart and the chart updates the UI, a SeriesCollection stores a collection of Series class, each Series has a Values property that implements IChartValues, this interface exposes the necessary methods to track Values property changes, so when you add/remove a value to series property, IChartValues will notify the chart so it can update the UI again. maybe you are asking yourself, what will happen if I add 1,000 point in 100 ms, would that slow down my application? No, LiveCharts implements a concept that we call DataSteps, a DataStep is a time interval that LiveCharts listens for data changes, when this interval ends then LiveCharts fires the update. This results in performance stable and totally automatic charts.
